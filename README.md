# 🔍 DevReviewStudy20

論理的思考と実装の妥当性を磨くための、20問の技術レビュー・シミュレーター。

## 🚀 プロジェクトの目的
本アプリは、プログラム内の非効率な実装や論理的な欠陥を特定し、ソフトウェア設計の原則に基づいた「適正かつ標準的な構成」へ修正するスキルを養うための学習ツールです。

## ✨ 特徴
- **20の実践的な検証課題**: データの不変性、リソース取得の効率化、例外処理の網羅性など、汎用的な20項目を厳選。
- **インタラクティブな判定機能**: 修正したソースコードが論理的要件を満たしているかを即座に検証。
- **本質的な技術解説**: 特定の製品や環境に依存しない、設計意図と論理的妥当性に焦点を当てた詳細なフィードバック。
- **品質指標の可視化**: 検証の精度を数値化し、客観的なコードレビュー能力の向上を支援。

## 🛠️ 技術構成
- **Frontend**: React (Vite)
- **Styling**: Tailwind CSS / Framer Motion
- **Design Philosophy**: 読みやすく、メンテナンスしやすいコード設計の追求

## 🤝 Credits & AI Collaboration
本アプリの演習問題および解説は、AI（Google Gemini）との共創によって制作されました。

- **問題の構成**: ソフトウェア設計のベストプラクティスに基づき、AIがシナリオを生成。
- **今後の展望**: 本アプリは学習コンセプトの提示を主眼としています。各問題の論理検証ロジックについては、今後さらに厳密なテストと改善を継続していく予定です。
